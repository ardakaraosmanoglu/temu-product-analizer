# Privacy Policy

Fabric Finder Extension ("this Extension") is committed to protecting your privacy.

## Data Collection

This Extension:
- **Does NOT collect** any personally identifiable information
- **Does NOT track** your browsing history
- **Does NOT sell or share** any user data to third parties

## Local Storage

All data (saved products, API key settings) is stored **locally in your browser** using `chrome.storage.local`. This data is never transmitted to any external servers except:
- **OpenRouter API** (for AI analysis) - only product metadata (title, material, composition, quality grade) is sent

## API Usage

When you use the AI analysis feature, product information is sent to OpenRouter API to generate fabric insights. This processing is solely for the AI analysis feature and is not used for any other purpose.

## Permissions Used

- `activeTab` - To read product page content when you click the extension icon
- `storage` - To save your favorite products locally in your browser

## Contact

For privacy concerns or questions, please open an issue on GitHub.
